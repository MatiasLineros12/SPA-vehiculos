# SPA-vehiculos
Single Page Application para la gestión de vehículos.
  - Inicio de sesión JWT
  - Almacenamiento de información en local storage
  - Gestión de vehículos
  - Consumo de API

Tecnologías de software:
  - PHP Laravel 6
  - AngularJS 11
  - Api RESTful
  - Json
  - Bootstrap 
  - MySQL
  - HTML

date dev: 2020
